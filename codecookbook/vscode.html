My Typical Initial Setup

Before installing Visual Studio Code, I found that things go smoother if I install a few things, first:

  1. Git Client

    • Go to https://git-scm.com/ to download the client that works well with Visual Studio Code.
    • Download GitHub Desktop from https://desktop.github.com/ to make it easier to create and manage repositories that are cloned locally from Git.

  2. Microsoft SDKs

    Desired .NET Core and .NET Framework versions can be downloaded from https://www.microsoft.com/net/download/visual-studio-sdks.

  3. Node.js

    The installer for node.js can be obtained from https://nodejs.org.

  4. Visual Studio Code

    Installer can be downloaded from https://code.visualstudio.com/download.

  5. Configure global git settings for credential helper and email address

    For Windows installations, it's probably best to use the Windows Credential Store to store your Git login credentials, so you don't have to enter your username and password every time you synchronize with the remote repository. Click here for information about configuring the credential helper
